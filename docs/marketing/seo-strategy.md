## Technical SEO checklist

- [ ] sitemap.xml with hreflang per language
- [ ] robots.txt allow all + sitemap reference
- [ ] llms.txt (LLM content map)
- [ ] OpenGraph + Twitter Cards per page
- [ ] JSON-LD: Organization, SoftwareApplication, Offer, FAQPage, BreadcrumbList
- [ ] Canonical URLs
- [ ] LCP < 1.5s, INP < 200ms, CLS < 0.05
- [ ] Submit to: Google Search Console, Yandex Webmaster, Bing, IndexNow
