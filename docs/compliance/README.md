# Compliance

Clary v2 is designed to meet:

- **Uzbekistan Law 547-son** on personal data (Persdata)
- **GDPR** (EU General Data Protection Regulation)
- **Russia 152-FZ** (for CIS customers)
- **HIPAA-compatible** healthcare safeguards (not formally certified yet)
