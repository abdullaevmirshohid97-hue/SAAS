# Persdata operator registration (Uzbekistan)

Under Uzbekistan Law No. ORQ-547 (2019) on Personal Data, operators must register with the authorized state body (Uzkomhurriyat / Uzbek State Personalization Center).

## Procedure

1. Company legal entity must be established in Uzbekistan (or have a representative office)
2. Within **one month** of starting data processing, file registration via <https://my.gov.uz>
3. Declare:
   - Company name, address, tax ID
   - Purpose of processing (healthcare SaaS)
   - Categories of data (name, phone, ID number, medical records)
   - Data subjects (clinic staff, patients)
   - Storage location (server in / outside Uzbekistan)
   - Cross-border transfer details
   - Security measures

## Status

- [ ] Legal entity established
- [ ] Registration filed
- [ ] Registration number received
- [ ] DPO (Data Protection Officer) designated

## Notes

- Failure to register is subject to fines up to 100x minimum wage
- The registration is public; list will include Clary LLC
