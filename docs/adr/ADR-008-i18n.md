# ADR-008: i18n — 7 languages + dual-script Uzbek

- Status: Accepted

## Decision

7 locales with ICU MessageFormat:

| Locale   | Script  | Scope                             |
|----------|---------|-----------------------------------|
| uz-Latn  | Latin   | Uzbekistan — default              |
| uz-Cyrl  | Cyrillic| Uzbekistan — older users          |
| ru       | Cyrillic| Russia + Uzbekistan (widely spoken)|
| kk       | Cyrillic| Kazakhstan                        |
| ky       | Cyrillic| Kyrgyzstan                        |
| tg       | Cyrillic| Tajikistan                        |
| en       | Latin   | International + developers        |

- Keys stored in `packages/i18n/locales/<locale>.json`
- Astro routing: `/uz`, `/uz-kr` (Cyrl), `/ru`, `/kk`, `/ky`, `/tg`, `/en`
- Machine translation (DeepL) seeded; human review for production
- RTL not required for these locales

## Consequences

- ~40,000 translation keys total
- One-time human translation cost: ~$2,500
- Language toggle switches both script and locale
