# Mobile assets

Add the following PNGs here:

- `icon.png` — 1024x1024 app icon
- `splash.png` — 1242x2436 splash image
- `adaptive-icon.png` — 1024x1024 Android adaptive icon foreground
- `favicon.png` — 48x48 web favicon
- `notification-icon.png` — 96x96 monochrome notification icon

Use the Clary brand palette (primary #2563EB, background #0A0A0A).
